# 🌸 Iris Flower Classification - ML Model Deployment with Flask & Render

A complete Machine Learning deployment project using the Iris dataset. This project demonstrates how to train a machine learning model, build a web application using Flask, and deploy it on Render for public access.

## 🚀 Project Overview

This project predicts the species of an Iris flower based on four input features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The trained model is integrated into a Flask web application and deployed on Render, allowing users to make predictions through a simple web interface.

## 🎯 Project Objectives

- Train a Machine Learning classification model using the Iris dataset.
- Save the trained model using Pickle.
- Build a user-friendly web application with Flask.
- Deploy the application online using Render.
- Demonstrate end-to-end Machine Learning deployment skill

## 📊 Dataset

The project uses the famous Iris Flower Dataset, which contains 150 samples belonging to three species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica
